=================================================================================
      Mandelbrot fractal generator --- ARM Neon implementation
=================================================================================


Compilation
------------------------------------------------------------------------

Simply type ``make``, program ``fractal`` will be created.

``./fractal fpu`` runs a FPU procedure
``./fractal neon`` runs an ARM Neon  procedure

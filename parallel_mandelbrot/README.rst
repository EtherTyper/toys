================================================================================
                    Parallel Mandelbrot set generator
================================================================================

Multithread generator of `Mandelbrot set`__ images.
On system with 2 CPUs speedup is around 1.95 --- 1.99.

__ http://en.wikipedia.org/wiki/Mandelbrot_set
